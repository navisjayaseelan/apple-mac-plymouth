**How to install (from file manager)**
* Download apple-mac-plymouth.tar.xz file
* Extract here
* Open folder apple-mac-plymouth
* Run install.sh

**How to install (from terminal)**
* Download apple-mac-plymouth.tar.xz file
* Extract here
* Open folder apple-mac-plymouth
* Right click (on an empty area)
* Click open in terminal
* Type command bellow
* chmod 777 install.sh
* sudo ./install.sh 
* Done

**Tested on**
* Ubuntu 18.04 LTS
* Xubuntu 18.04
* Kubuntu 18.04
* Linux Mint 19
